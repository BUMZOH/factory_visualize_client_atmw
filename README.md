# 光和工業 工場見える化 クライアントアプリ

## 機能



## 仕様
- アプリ名: "factory_visualize_client_atmw"
- 工場の稼働データを見える化する
- 元データはSQLiteファイル(main_factory_production_data.db)
- pywebviewを使ったSPA型アプリ
- ページは以下の3ページ
-  ① 設備別詳細データ（設備1台に対して1日の稼働状況を詳しく表示）
-  ② 全設備稼働データ（全設備に対して1日の稼働状況を表示）
-  ③ 設備別稼働データ推移（設備1台に対して指定した期間の稼働状況を表示）
-  ページ上部に上の①～③のページ切り替えボタンを配置
-  画面はFull HD (1920x1080)での利用を想定
